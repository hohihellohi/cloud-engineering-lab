# ☁️ Cloud Engineering Lab

안녕하세요. 클라우드 인프라와 Kubernetes 기반 서비스 플랫폼 운영에 관심을 가지고 있는 Cloud / DevOps Engineer 김종호입니다.

AWS, OCI, Kubernetes, Terraform, CI/CD, Observability 영역을 중심으로 인프라 설계, 구축, 운영, 자동화 경험을 정리하고 있습니다.  
이 저장소는 단순한 개념 정리가 아니라, 실제 테스트 과정에서 사용한 Terraform 코드, Kubernetes Manifest, Shell Script, 설정 파일, 테스트 결과를 함께 관리하기 위한 기술 기록 저장소입니다.

---

## 👨‍💻 About Me

저는 클라우드 인프라 엔지니어로서 AWS, OCI, Kubernetes 기반의 인프라 설계, 구축, 운영, 자동화 업무를 수행하고 있습니다.

주요 관심 분야는 다음과 같습니다.

- ☁️ AWS / OCI 기반 클라우드 인프라 설계 및 운영
- ☸️ Kubernetes / EKS / OKE 기반 컨테이너 플랫폼 운영
- 🧱 Terraform / CloudFormation 기반 Infrastructure as Code
- 🚀 Jenkins / GitHub Actions / GitLab / ArgoCD 기반 CI/CD 자동화
- 📈 OpenTelemetry / Prometheus / Grafana / ClickHouse 기반 Observability
- 🐧 Linux, Network, Middleware 운영 및 트러블슈팅
- 🛠️ 운영 자동화 및 장애 대응 프로세스 개선

이 저장소에는 제가 학습하거나 테스트한 내용을 정리하는 것뿐만 아니라, 실제 운영 환경에서 마주칠 수 있는 문제를 어떻게 분석하고 해결할 수 있는지에 대한 기록을 함께 남기고 있습니다.
