## 👨‍💻 About Me

클라우드 인프라 엔지니어로서 AWS, OCI, Kubernetes 기반 인프라 설계/구축/운영/자동화 업무를 수행하고 있습니다.

주요 관심 분야:

- ☁️ AWS / OCI 기반 클라우드 인프라 설계 및 운영
- ☸️ Kubernetes / EKS / OKE 기반 컨테이너 플랫폼 운영
- 🧱 Terraform / CloudFormation 기반 Infrastructure as Code
- 🚀 Jenkins / GitHub Actions / GitLab / ArgoCD 기반 CI/CD 자동화
- 📈 OpenTelemetry / Prometheus / Grafana / ClickHouse 기반 Observability
- 🐧 Linux, Network, Middleware 운영 및 트러블슈팅
- 🛠️ 운영 자동화 및 장애 대응 프로세스 개선
